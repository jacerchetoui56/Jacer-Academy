HTML

HTML Introduction

HTML stands for Hyper Text Markup Language,it is the standard markup language for creating Web pages . HTML describes the structure of a Web page. It consists of a series of elements .These elements tell the browser how to display the content.

HTML Example

A sample HTML example :

Try It !DOCTYPE html
html
head
titleExampletitle
head
body
h1this is my first Headingh1
pthis is my first paragraph.p
body
html

Example Explained :

  • <!DOCTYPE html> defines that this document is an HTML5 document.
  • <html> this element is the root element of an HTML page.
  • <head> contains meta information about the HTML page.
  • <title> specifies a title for the HTML page.
  • <body> defines the document's body, and is a container for all the contents, such as headings, paragraphs, images, tables, etc.
  • <h1> defines a large heading.
  • <p> defines a paragraph.
Note : HTML tags are not case sensitive: <P> means the same as <p>.

HTML Page Structure

HTML Elements

The HTML element is everything from the start tag to the end tag:

Examples :

h4this is a smaller headingh4
pthis is a paragraphp
Note : Some HTML elements have no content (like the <br> element : line break).

HTML Comments

HTML comments are not displayed in the browser, but they can help document your HTML source code. Syntax :

<!-- This is a comment -->

HTML Most Common Tags

HTML Headings :

HTML headings are defined with the <h1> to <h6> tags.

Example :

h2this is heading 2h2

HTML Paragraphs :

HTML paragraphs are defined with the <p> tag.

Example :

HTML paragraphs are defined with the <a> tag.

Example :

HTML lists allow web developers to group a set of related items in lists.

  • An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

HTML Images :

HTML images are defined with the <img> tag.

Example :

img src="./images/img1.jpg" alt="an image" width="100" height="120"

Explanation :

  • src is the source file.
  • alt is the alternate text for an image.
  • width and height are the properties of the image.

HTML div tag :

  • The <div> tag defines a division or a section in an HTML document.

  • The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

  • The <div> tag is easily styled by using the class or id attribute.

Notes :
- By default, browsers always place a line break before and after the div element.
- we will learn about HTML Classes and ID later

HTML Attributes

Properties :

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes usually come in name/value pairs like: name="value"

Most Common Attributes :

  • The href attribute of <a> specifies the URL of the page the link goes to.
  • The src attribute of <img> specifies the path to the image to be displayed.
  • The width and height attributes of <img> provide size information for images
  • The alt attribute of <img> provides an alternate text for an image.
  • The style attribute is used to add styles to an element, such as color, font, size, and more.
    p style="color:blue;"this is a blue paragraphp
  • The lang of <html> tag declares the language of the Web page.
    !DOCTYPE html
    html lang="en"
    body

    body
    html
  • The title attribute defines some extra information about an element.
    p title="I'm a paragraph"this is a paragraphp

HTML Class :

CSS

CSS Introduction

CSS stands for Cascading Style Sheets It is the language we use to style an HTML document. CSS describes how HTML elements should be displayed. CSS saves a lot of work. It can control the layout of multiple web pages all at once

CSS Syntax

A CSS rule consists of a selector and a declaration block.

Example :

p{
  color:red;
  font-size:1.5rem;
}

Explanation :

  • p is a selector in CSS (it points to the HTML element you want to style: <p>).
  • color is a property, and red is the property value
  • font-size is a property, and 1.5rem is the property value

CSS Comments

CSS comments are not displayed in the browser, but they can help document your source code.

A CSS comment is placed inside the <style> element, and starts with /* and ends with */

Example :

/* this is a comment */
h2{
  color:blue; /* text color set to blue */
  text-align:center; /* text is positionned in the center */
}

The internal style is defined inside the <style> element, inside the <head> section.

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

CSS element selectors :

The element selector selects HTML elements based on the element name.

Example :

p{
  color:red;
  font-size:1.5rem;
}

CSS id selector :

The id selector uses the id attribute of an HTML element to select a specific element.

Example :

#heading1 {
  color:red;
  font-size:1.5rem;
}
Note : An id name cannot start with a number!

CSS class selector :

The class selector selects HTML elements with a specific class attribute.

Exmaple :

.paragraph1 {
  color:green;
  font-weight:bold;
}

CSS Universal selector :

The universal selector * selects all HTML elements on the page.

Example :

* {
  margin:0;
  padding:0;
  box-sizing:border-box;
}

CSS grouping selector :

The grouping selector selects all the HTML elements with the same style definitions.

Example :

Try It p , #elt , .text{
  color:red;
  border:solid 1px grey;
}
